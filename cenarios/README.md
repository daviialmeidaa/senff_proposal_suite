# Cenarios

Esta pasta centraliza as especificações de cenários de validação das esteiras de proposta.
Cada arquivo Markdown descreve regras, comportamentos esperados e critérios de aprovação/rejeição para uma etapa específica.
O conteúdo aqui orienta a geração automática das validações no motor de execução da suite.

## Estrutura

```
cenarios/
  validacoes_esteira/     # Regras por etapa (stage.code) de cada esteira
    protheus/             # Etapas da esteira Protheus
    ...
```
